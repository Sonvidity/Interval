# **App Name**: Interval: Adaptive Vehicle Maintenance

## Core Features:

- Vehicle Database: Centralized, searchable database containing OEM service intervals for various car makes and models.
- User Garage: Allows users to add and manage their vehicles, including modifications, driving style, and service history.
- Adaptive Interval Engine: Calculates dynamic service intervals based on the user's vehicle modifications (ModFactor) and driving style (DriveFactor).
- Engine Swap Support: Allows users to track engine and chassis kilometers separately for engine-swapped vehicles.
- Service History Log: Maintained history of all vehicle services, including date, mileage, service type, cost, and items done.
- Calculation Explanation: Provides a transparent breakdown of how service intervals are calculated, showing the base interval, modification factor, and driving style factor.

## Style Guidelines:

- Primary color: Dark Teal (#004040) to evoke a sense of reliability and mechanical precision in a dark mode setting.
- Secondary color: Dark Gray (#333333) for the background to provide a modern dark mode look.
- Accent color: Electric Cyan (#7DF9FF) to highlight interactive elements and important information, ensuring visibility in dark mode.
- Headline font: 'Space Grotesk', a sans-serif font for a modern and technical aesthetic, optimized for dark mode readability.
- Body font: 'Inter', sans-serif font for the body text to ensure optimal readability in dark mode.
- Use of outlined icons with subtle, glowing animations on hover to enhance user experience in dark mode.
- Card-based layout with rounded corners and subtle shadows for a clean and organized presentation of information in a dark mode environment.
